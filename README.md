<h2>Prerequisites</h2>
<ul>
<li>node.js</li>
<li>nodemon</li>
</ul>
<h2>Installation</h2>
<ol>
<li>
Clone the repository to your github and download the files.
</li>
<li>
Open the terminal, change directory into the main folder.
</li>
<li>
In the terminal, run "npm i" to install the necessary packages.
</li>
</ol>
